# 🚀 Capítulo 3: Fundamentos y Arquitectura de Spring Boot

Bienvenido al repositorio del **Capítulo 3**. En esta sección, evolucionamos desde la configuración básica hacia la construcción de aplicaciones robustas, escalables y listas para producción.

## 📝 Contenido del Capítulo
El código de este directorio cubre los siguientes ejes temáticos:

* **3.1 Arquitectura:** Inyección de dependencias y el ecosistema Spring.
* **3.2 MVC & Thymeleaf:** Renderizado de vistas dinámicas en el servidor.
* **3.3 APIs REST:** Construcción de servicios finales con **DTOs** (Data Transfer Objects).
* **3.4 Routing:** Manejo avanzado de `@PathVariable` y `@RequestParam`.
* **3.5 SpEL:** Uso de *Spring Expression Language* para configuraciones dinámicas.
* **3.6 Deploy:** Empaquetado con Maven para despliegue en **Ubuntu Server**.

## 🚀 Instrucciones de Ejecución
Para probar los ejemplos de este capítulo en tu entorno local o servidor:

1. **Clonar el repositorio:**
   ```bash
   git clone [https://github.com/tu-usuario/tu-repositorio.git](https://github.com/tu-usuario/tu-repositorio.git)
   cd capitulo3/Spring
2. **Compilar y copiar:**
./mvnw spring-boot:run

3. **Acceso:**
Abre tu navegador en http://localhost:8080 para las vistas de Thymeleaf o usa Postman/Insomnia para las APIs REST en http://localhost:8080/api/...
